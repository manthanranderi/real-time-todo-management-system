
## Introduction

The Professional Todo List Application is a web-based task management system developed using Node.js, Express.js, and EJS.  

This application helps users organize and manage their daily tasks in a simple and efficient way. The project includes a modern professional UI with a blue gradient background and responsive design.

Users can add, update, and delete tasks easily through a clean dashboard interface.

---

# Project Objectives

- To develop a task management application using Node.js and Express.js
- To perform CRUD operations effectively
- To design a professional and responsive user interface
- To understand server-side rendering using EJS templates

---

# Installation Process

## Step 1: Download or Clone the Project

```bash
git clone <repository-url>
```

---

## Step 2: Open the Project Folder

```bash
cd Todo-List
```

---

## Step 3: Install Dependencies

```bash
npm install
```

---

## Step 4: Run the Application

```bash
node app.js
```

---

# Server Information

After running the server, open the following URL in your browser:

```bash
http://localhost:8000
```

# Working of the Application

1. The user opens the dashboard.
2. The user can add a new task using the form.
3. All tasks are displayed in the task table.
4. The user can edit or delete tasks anytime.
5. Task status and priority help manage workflow efficiently.
